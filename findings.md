## Tech Stack Decisions
- **Core:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS (responsive, mobile-first)
- **Export:** Static Export (`output: 'export'`) for simple hosting.
- **Animations:** `framer-motion` for complex UI interactions, `aos` or custom CSS for scroll reveals.
- **Icons:** `lucide-react` (clean, modern, consistent).
- **Data Source:** JSON files in `data/` mirroring `gemini.md` schema.

## Integrations
- **Maps:** Google Maps Embed (Iframe).
- **Forms:** Formspree (or similar static form handler).
- **Chatbot:** Placeholder widget.

## Constraints
- **Design:** Dark navy (#001F3F) & Fiery orange (#FF4500).
- **Fonts:** Poppins (Headers), Inter (Body).
